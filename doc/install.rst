============
Installation
============

Install from a Release
======================

Statically-linked binaries and Debian packages are available on the
`releases page`_.

Install with Cargo
==================

You can also install souffle-lint via Cargo and crates.io:

.. code-block:: bash

   cargo install souffle-lint

=============================
